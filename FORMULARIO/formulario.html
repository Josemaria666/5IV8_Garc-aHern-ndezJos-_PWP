Formulario

Todos los formularios en la web, manejan metodos, para el envio de diferentes peticiones que realiza el usuario consdierando la arquitectura cliente servidor, el manejo de la soeticiones se hace por medio del protocoll http y actualmente por https cada peticion debe de ser atendida en un tiempo de respuesta conforme al modelo TCP/IP. Los metydos principales de los formularios son:

    • GET:

      Este metodo se ecnarga de enviar la inofrmacion del formulario por medio de la salida estandar (conocida como url), del navegador esto propicia que puedan existir inyecciones de codigo realizando violaciones a la seguridad del sistema

      POST:

      Este metodo se encarga de envbiar la informacion para poder resgistrar un cambio o un nuevo registro dentro del sistema enviandola igual, por el metodo estandar, pero mediante un ocultamiento, esto significa que no es visible para el horario lo que nos ofrece un mayor grado de seguridad en el manejo de la informacion


    • En resumen los metodos son:

    • Get: Sirve para enviar y realizar operaciones de lectura o consulta
      Post: Sirve para enviar info de un recurso al servidor, con el fin de registrarlo
      Delete: Sirve para eliminar un recurso específico dentro del servidor obteniendo una respuesta de actualización
      Put: Sirve para almacenar la info de un recurso particular y conectarse al servicio paravrealizar la actyalización
      Head: Sirve para regresar el codigo de puesta de cualquier cabecera HTTP asociada a un recurso
